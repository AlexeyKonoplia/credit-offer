## Итоговые результаты

Данный репозиторий не отражает полностью весь ход решения. Объем проверенных подходов сильно больше и просто будет нечитаемым.


| Вариант | OOF AUC | Public LB |
|---|---|---|
| одиночный CatBoost (бейзлайн) | 0.834 | — |
| ансамбль 3 моделей, ранговый бленд | 0.8366 | 0.7688 |
| **ансамбль + seed-bagging** | 0.8374 | 0.7698 |

Тест продолжает train по времени. После ряда итераций отказались от target encoding
и тяжёлых произведений признаков (поднимали OOF, но не лидерборд), перешли на
ранговый бленд трёх разных бустингов и добавили seed-bagging.

## Подход

- **Признаки**: временные (включая порядковый день — он ловит временной тренд),
  наценка над ставкой ЦБ, разности лимитов и активности, счётчик пропусков. От
  target encoding и тяжёлых произведений отказались — поднимали OOF, но не лидерборд.
- **Модели**: ансамбль из 3 разных бустингов — LightGBM + CatBoost + XGBoost — с
  нативной обработкой категорий.
- **Ансамблирование**: ранговый бленд (у моделей разные шкалы, для AUC важен только
  порядок) + seed-bagging для снижения дисперсии разбиения.
- **Валидация**: 5-fold stratified — обобщает лучше хронологической несмотря на
  временной сдвиг теста.


## Запуск

Положить рядом с ноутбуком: `train_apps.csv`, `test_apps.csv`

```bash
pip install -r requirements.txt
```
 - Далее "Run All" в solution.ipynb


Полный прогон (3 модели × 5 фолдов × 3 сида) занимает порядка 40 минут на CPU.
Чтобы быстрее — оставить один сид в `BAG_SEEDS`.
