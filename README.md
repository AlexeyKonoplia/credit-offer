# Отклик на кредитный оффер

Решение задачи предсказания вероятности согласия корпоративного клиента на кредитное
предложение. Метрика — **ROC-AUC**.

## Результаты

| Вариант | OOF AUC | Public LB |
|---|---|---|
| одиночный CatBoost | 0.834 | - |
| ансамбль 7 моделей (простое среднее) | 0.8367 | 0.7667 |
| **ансамбль + seed-bagging** | 0.8376 | 0.7672 |

## Подход

- **Признаки**: временные (включая порядковый день — он ловит временной тренд),
  флаги пропусков, доменные interactions (произведения/отношения/разности),
  out-of-fold target encoding категорий.
- **Модели**: ансамбль из 7 моделей — LightGBM ×3, CatBoost ×2, XGBoost ×2 — на двух
  наборах признаков (с target encoding и без). Базовые (lr=0.02) и консервативные
  (lr=0.01, сильная регуляризация).
- **Ансамблирование**: простое среднее (устойчивее взвешенного и стекинга при
  высокой корреляции моделей) + seed-bagging для снижения дисперсии разбиения.
- **Валидация**: 5-fold stratified — обобщает лучше хронологической несмотря на
  временной сдвиг теста.
